First write / 19.4.2023

This is the first write on this website, idk what to talk about, so I'll just tell about my current linux build.

I've been using linux for 2 years now and I have been switching between many different distros (fedora -> debian -> endeavour -> arch -> void -> artix -> ...), but currently I've ended up back using EndeavourOS with i3wm.

I like to keep my system minimalistic, so I have only modified the default endeavour i3 config a little bit to fit my likings. I don't use polybar, waybar or any other bloat, I only use the default i3bar like a chad. Really just keep it simple.

My old i3wm rice is public on my GitHub page (click here.), but maybe I'll release my current one some day too. Maybe after I switch distros again and get bored again of this one. (nixos next?)
